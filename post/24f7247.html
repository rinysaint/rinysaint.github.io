第五步：nginx配置

server {
    listen 80 ;

    root /home/git/www/hexo;

    server_name hexo.ixiewei.com;                                 

    location / {                                                             

        root /home/git/www/hexo;

        if (-f $request_filename) {

            rewrite ^/(.*)$  /$1 break;

        }

    }                                                  
}

此处保证nginx运行账户，拥有/home/git/www/hexo各级目录权限，否则可能出现403
